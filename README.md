
1) What is the difference between null and undefined?
১. "Null" হলো একটি value, যা নিজের ইচ্ছায় দেওয়া হয়, কিন্তু "Undefined" by default ব্রাউজার থেকে আসে যখন কোন ফাংশন এর মান না দেওয়া হয়। 

2) What is the use of the map() function in JavaScript? How is it different from forEach()?
২. map() এমন একটি ফাংশন যা array কে পরিবর্তন করে নতুন নতুন array সৃষ্টি করে কিন্তু forEach() শুধু প্রতিটা element কে ভাগ করে উপস্থাপন করে।


3) What is the difference between == and ===?
৩. == দিয়ে Value compare করে কিন্তু Type convert করে অনেকসময় সমান সমান দেখিয়ে দেয়, কিন্তু ===  স্ট্রিকলি Type and value চেক করে। 


4) What is the significance of async/await in fetching API data?
৪. async/await দিয়ে API এর ডাটা আসার জন্য ওয়েট করা হয়, যাতে কোড স্পষ্ট আর পেইজ ব্লাংক হয়ে না যায়। 


5) Explain the concept of Scope in JavaScript (Global, Function, Block).
৫. Global Scoop: গ্লোবাল স্কোপ দিয়ে যেকোন জায়গা থেকে থেকে ভ্যালু ক্যাচ করা যায়। 
Function Scoop: শুধুমাত্র ফাংশন এর ভেতরের ভ্যালু এক্সেস করা যাবে। 
Block Scoop: শুধু মাত্র {} এর ভিতরে কাজ করে।
